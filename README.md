Smart Online Examination & Assessment System

<img width="1881" height="903" alt="image" src="https://github.com/user-attachments/assets/3853cf6e-9e82-461b-a559-c70477dcd701" />
Smart Online Examination & Assessment System is a modern web-based examination platform developed using the MERN Stack. It enables educational institutions, coaching centers, and organizations to conduct secure online examinations, automatically evaluate objective answers, manage question banks, analyze student performance, and generate detailed reports.
The platform supports multiple user roles, intelligent question randomization, coding assessments, real-time analytics, role-based dashboards, and comprehensive reporting to simplify the complete examination lifecycle.
